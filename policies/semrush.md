# Policy: SEMrush — правила безопасного использования

## Контекст

SEMrush — платный сервис SEO-аналитики ($200+/мес). Аккаунт командный (не только наш).
Доступ через Pinchtab (headless Chrome). Цель — только сбор данных (keyword research, export CSV).
Любая ошибка может затронуть чужие проекты/настройки.

## КАТЕГОРИЧЕСКИЙ ЗАПРЕТ — никогда не делать

1. **Не удалять ничего** — проекты, списки, кампании, отчёты, ключи
2. **Не редактировать существующее** — чужие проекты, настройки, трекинги, Position Tracking
3. **Не менять настройки аккаунта** — биллинг, пользователи, API ключи, интеграции
4. **Не создавать проекты** — они занимают лимитированные слоты (5–40 в зависимости от плана)
5. **Не подключать домены** — Domain Overview/Site Audit/Position Tracking могут создать привязку
6. **Не нажимать "Delete", "Remove", "Unsubscribe", "Cancel"** — вообще никогда
7. **Не заходить в разделы**: Settings, Account, Billing, Users, API, Subscription
8. **Не использовать Keyword Strategy Builder** — он создаёт списки, расходует лимиты, может задеть существующие списки

## РАЗРЕШЁННЫЕ ИНСТРУМЕНТЫ — только read-only поиск

### Keyword Magic Tool (основной инструмент)
- Вводить seed-ключ в поисковую строку
- Выбирать страну/базу данных (Brazil, Mexico)
- Применять фильтры (Volume, KD, Intent, Include/Exclude words)
- Менять match type (Broad, Phrase, Exact, Related)
- Сортировать таблицу по колонкам
- Экспортировать результаты (CSV/XLSX) — это безопасно, это download
- Переключать группы ключевых слов в левом сайдбаре

### Keyword Overview (вспомогательный)
- Вводить ключевое слово
- Выбирать страну
- Читать метрики (volume, KD, CPC, intent)
- Смотреть SERP analysis

### Organic Research (вспомогательный, для конкурентов)
- Вводить домен конкурента (duolingo.com, talkpal.ai и т.д.)
- Выбирать страну/базу
- Смотреть Top Keywords, Top Pages, Competitors
- Экспортировать результаты

## ПРАВИЛО СОМНЕНИЙ

Если я вижу кнопку/элемент и НЕ УВЕРЕН на 100%, что это безопасно:
1. **НЕ кликаю**
2. **Делаю snapshot_interactive** — получаю список элементов
3. **Описываю пользователю** что вижу и что хочу нажать
4. **Жду подтверждения** прежде чем кликать

Это правило действует ВСЕГДА при первом использовании любого нового элемента UI.

## ПРАВИЛО НАВИГАЦИИ

- Перемещаться только через прямые URL (navigate) — не через кнопки сайдбара, если не уверен куда они ведут
- Основные рабочие URL:
  - `https://www.semrush.com/analytics/keywordmagic/` — Keyword Magic Tool
  - `https://www.semrush.com/analytics/keywordoverview/` — Keyword Overview
  - `https://www.semrush.com/analytics/organic/overview/` — Organic Research

## ПРАВИЛО ЭКСПОРТА

- Экспорт (CSV/XLSX download) — безопасен, это просто скачивание файла
- Кнопка Export обычно находится вверху справа над таблицей
- Если сервер предлагает "Export to PDF" или "Send to..." — переспросить пользователя
- Скачанные файлы появятся в домашней директории или ~/Downloads

## ПРАВИЛО ЛИМИТОВ

- SEMrush имеет дневные лимиты запросов (зависит от плана)
- Не делать больше 20 поисковых запросов подряд без паузы
- Если сервис показывает warning о лимитах — остановиться, сообщить пользователю
- Записывать использованные запросы в notes

## ЧТО ДЕЛАТЬ ПРИ ОШИБКЕ

Если случайно открылась не та страница, нажалось не то:
1. **НЕ пытаться "исправить" кликая дальше** — это может усугубить
2. **Сделать snapshot** — зафиксировать что произошло
3. **Сообщить пользователю** — описать что случилось
4. **Ждать инструкций**
